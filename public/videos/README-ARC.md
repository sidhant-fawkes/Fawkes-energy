# FawkesArc Video Files Required

The following video files need to be created for the FawkesArc product:

- `fawkesarc.webm` - WebM format video (primary)
- `fawkesarc.mp4` - MP4 format video (fallback)

These should follow the same pattern as other product videos (fawkeslink, fawkescore, fawkesloop).

Optional mobile versions:
- `fawkesarc-mobile.webm`
- `fawkesarc-mobile.mp4`
